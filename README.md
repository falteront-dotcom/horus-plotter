# 🖊️ Horus Plotter

> Загрузи картинку или текст — плоттер нарисует твоим почерком

**Horus Plotter** — программа для управления пишущим плоттером (роботом-каллиграфом). Два режима: конвертация **текста** в рукописный конспект и конвертация **изображений** в художественную графику.

## 🎯 Возможности

### 📝 Текстовый режим (конспекты лекций)
- 5 рукописных шрифтов (личный курсив, прописи, ГОСТ, элегантный, жирный)
- Тетрадная разметка: красные поля, строки
- Автооптимизация размера шрифта под тетрадь
- Разбивка на страницы с превью
- Экспорт в G-код для любого GRBL-плоттера

### 🖼 Режим изображений
- 17 художественных стилей (штриховка, спираль, точечная, гравюра, соты...)
- Авто-инверсия для тёмного фона
- Яркость, контраст, размытие
- Превью в реальном времени

### 🔮 Нейро-зрение
- Автоопределение полей и строк по фото тетради
- Автоподбор размера шрифта и интервала
- Коррекция смещения и поворота

## 🚀 Быстрый старт

### Установка
```bash
git clone https://github.com/falteront-dotcom/horus-plotter.git
cd horus-plotter
pip install flet pillow pyserial numpy scipy python-docx pymupdf
```

### Три способа запуска

#### 1. GUI (рекомендуется)
```bash
python app.py
```
Две вкладки: 🖼 Изображение + 📝 Текст/Конспекты

#### 2. CLI (одна команда)
```bash
# Конвертировать текст → G-код с автооптимизацией
python plot.py лекция.txt --auto

# Выбрать шрифт и посмотреть статистику
python plot.py лекция.txt --font engineer_print --preview

# Сохранить в файл
python plot.py лекция.txt -o конспект.gcode

# Все опции
python plot.py --help
```

#### 3. Electron (альтернативный UI)
```bash
npm install
npm start
```

## 🎨 Шрифты

| Название | Стиль | Для чего |
|---|---|---|
| `semyon_cursive` | Личный курсив | Конспекты, письма |
| `school_script` | Школьные прописи | Аккуратные конспекты |
| `engineer_print` | ГОСТ / Технический | Чертёжное письмо |
| `elegant_italic` | Элегантный наклонный | Заголовки, открытки |
| `bold_round` | Жирный округлый | Титульные, обложки |

## 📂 Структура

```
horus-plotter/
├── app.py              # Flet GUI
├── plot.py             # CLI
├── core/
│   ├── fonts.py        # 5 шрифтов
│   ├── text_engine.py  # Текст → штрихи
│   ├── notebook.py     # Тетрадная разметка
│   ├── text_pipeline.py# Текст → G-код
│   ├── vision.py       # Нейро-зрение
│   ├── converters.py   # 17 стилей изображений
│   ├── gcode.py        # G-код + превью
│   ├── pipeline.py     # Изображение → G-код
│   └── paths.py        # Оптимизация путей (TSP+RDP)
├── tests/              # 172 теста ✅
├── index.html          # Electron UI
└── main.js             # Electron backend
```
